# Artist Portfolio Website

A simple single-page website for showcasing an artist's work.

## Usage

Open `index.html` in your web browser to view the site. Use the navigation in the top left to switch between sections: Hello!, Works, About, and Contact.

## Structure

- `index.html`: The only file needed. Contains the homepage with a "Hello!" greeting and navigation to Works, About, and Contact sections.

## Customization

- Edit `index.html` to update the greeting, artwork, biography, and contact details.
- Replace `placeholder.jpg` with actual images of your artwork.
- Modify the CSS in the `<style>` tag to change colors, fonts, layout, or spacing.

## Troubleshooting

- If images don't load, ensure the image files are in the same directory as `index.html` or update the paths accordingly.
- Use a local web server or open the file directly in a browser.
